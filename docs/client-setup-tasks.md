## 3. StripeとGCP（Googleログイン実装用）のアカウント招待

以下の手順でアカウントへの招待をお願いいたします。

### Stripe手順

1. [Stripeダッシュボード](https://stripe.com)右上の「設定（歯車アイコン）」をクリック
2. 「チーム」または「チームとセキュリティ」を選択
3. 「+ メンバーを追加」をクリック
4. 以下の情報を入力し、招待を送信してください

| 項目        | 値                        |
| --------- | ------------------------ |
| 招待メールアドレス | <hidenariyuda@gmail.com> |
| 役割（ロール）   | 開発者 (Developer)          |

### GCP手順

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. 上部のプロジェクト選択プルダウンから、対象のプロジェクトを選択（または「新しいプロジェクト」を作成）
3. 左側メニューの「IAMと管理」→「IAM」をクリック
4. 上部の「アクセス権を付与」（または「追加」）をクリック
5. 以下の情報を入力して保存してください

| 項目        | 値                        |
| --------- | ------------------------ |
| 新しいプリンシパル | <hidenariyuda@gmail.com> |
| ロールを選択    | 基本 → 編集者 (Editor)        |

## 5. 送信用メールアドレスについて

Supabaseのデフォルトドメインの場合、**1時間あたり3回まで**しかメール送信ができません。

Google Workspaceなどでカスタムドメインのメールアドレスをお持ちの場合、SMTPの設定をしていただき、以下の情報を共有いただけますでしょうか：

* 送信用メールアドレス
* SMTPパスワード（アプリパスワード）

### Google Workspaceでのアプリパスワード設定手順

以下のURLから設定可能です：

<https://myaccount.google.com/apppasswords>
